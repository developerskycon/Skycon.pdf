This method is used to convert a html file or content, into the pdf document.

Parameters
Parameters Description
Html If content of a html document in memory, wants to be converted to pdf then this parameter is used.
config is used for setting up to convert operation.
Please refer to PdfConfig for further information about configuration.
EventId is optional. This parameter is used for callback events. If this parameter is leave nothing then it is accepted as Now.Ticks
DontThrowEvent is optional. If events are not wanted to be thrown; must specify false. Otherwise parameter value is true by default.
