Parameters
Parameters Description
PdfData is the content to be converted in to image(s).
EventId is optional. This parameter is used for callback events. If this parameter is leave nothing then it is accepted as Now.Ticks
DontThrowEvent is optional. If events are not wanted to be thrown; must specify false. Otherwise parameter value is true by default.
xDpi, yDpi is optional. is the unit (dot per inch) of resolution for conversion operation.
