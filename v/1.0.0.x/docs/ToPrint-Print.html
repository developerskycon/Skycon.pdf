Parameters
Parameters Description
PdfData is the content to be printed.
DocumentName is the name of the document printed in the spool.
PrinterName is optional. is the name of the printer to be used for printing. Default value is the default-printer of the OS.
xDpi, yDpi is optional. is the unit (dot per inch) of resolution for printing operation.
