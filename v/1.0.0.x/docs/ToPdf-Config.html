ToPdfConfig is used for PDF convertions. All settings are shown below. Definition Namespace : Skycon.Pdf 
		public Class ToPdfConfig
	
		Public Class ToPdfConfig
Settings
General
NameTypeDescription
DocumentTitle StringThe title of the PDF document.
Resolution StringDeprecated, has no effect.
Collate BooleanShould the copies be collated?
Outline BooleanShould a outline (table of content in the sidebar) be generated and put into the PDF? Must be either "true" or false".
UseCompression BooleanShould we use loss less compression when creating the pdf file?
Output StringThe path of the output file, if "-" output is sent to stdout, if empty the output is stored in a buffer.
DumpOutline StringIf not set to the empty string a XML representation of the outline is dumped to this file.
CookiePath StringPath of file used to load and store cookies.
DPI LongDPI used when printing, like "80".
PageOffset StringA number that is added to all page numbers when printing headers, footers and table of content
Copies LongHow many copies should we print?. e.g. "2".
OutlineDepth LongThe maximal depth of the outline, e.g. "4".
MaxImageDPI LongThe maximal DPI to use for images in the pdf document.
ImageQuality LongThe jpeg compression factor to use when producing the pdf document, e.g. "92".
Page StringThe URL or path of the web page to convert, if "-" input is read from stdin.
TocXsl StringIf not empty this object is a table of content object, "page" is ignored and this xsl style sheet is used to convert the outline XML into a table of content.
UseExternalLinksBooleanShould external links in the HTML document, be converted into external pdf links?
UseLocalLinks BooleanShould internal links in the HTML document, be converted into pdf references?
ProduceForms BooleanShould we turn HTML forms into PDF forms? Must be either "true" or false".
PagesCount BooleanShould we count the pages of this document, in the counter used for TOC, headers and footers?
IncludeInOutlineBooleanShould the sections from this document be included in the outline and table of content?
PaperSize StringThe paper size of the output document, e.g. "A4".
PaperWidth StringThe with of the output document, e.g. "4cm".
PaperHeight StringThe height of the output document, e.g. "4cm".
Orientation StringThe orientation of the output document, must be either "Landscape" or "Portrait". Default value is "Portrait".
ColorMode StringShould the output be printed in color or gray scale, must be either "Color" or "Grayscale". Default value is "Color".
IsGrayscale BooleanShould the output be printed in color or gray scale. Default value is "Color".
MarginTop StringSize of the top margin. (exm 1in, 5cm etc.)
MarginRight StringSize of the right margin. (exm 1in, 5cm etc.)
MarginBottom StringSize of the bottom margin. (exm 1in, 5cm etc.)
MarginLeft StringSize of the left margin. (exm 60, 1in, 5cm etc.)

ToPdfConfig_Toc - Table of content configs.
NameTypeDescription
UseDottedLines BooleanShould we use a dotted line when creating a table of content?
CaptionText StringThe caption to use when creating a table of content.
ForwardLinks BooleanShould we create links from the table of content into the actual content?.
BackLinks BooleanShould we link back from the content to this table of content.
FontScale SingleHow much should we scale down the font for every toc level? E.g. "0.8".
Indentation StringThe indentation used for every table of content level, e.g. "2em".


ToPdfConfig_Web - Web page specific configs.
NameTypeDescription
Background BooleanShould we print the background?
LoadImages BooleanShould we load images?
EnableIntelligentShrinking BooleanShould we enable intelligent shrinkng to fit more content on one page? Has no effect for wkhtmltoimage.
EnableJavascript BooleanShould we enable javascript?
MinimumFontSize LongThe minimum font size allowed. E.g. "9"
PrintMediaType BooleanShould the content be printed using the print media type instead of the screen media type. Has no effect for wkhtmltoimage.
DefaultEncoding StringWhat encoding should we guess content is using if they do not specify it properly? E.g. "utf-8".
UserStylesheet StringUrl er path to a user specified style sheet.
EnablePlugins BooleanShould we enable NS plugins. Enabling this will have limited success.


ToPdfConfig_Load - Object loading configs.
NameTypeDescription
UserName StringThe user name to use when loging into a website.
Password StringThe password to used when logging into a website.
JsDelay LongThe mount of time in milliseconds to wait after a page has done loading until it is actually printed. E.g. "1200". We will wait this amount of time or until, javascript calls window.print().
ZoomFactor DecimalHow much should we zoom in on the content? E.g. "2.2"
RepeatCustomHeaders BooleanShould the custom headers be sent all elements loaded instead of only the main page?
BlockLocalFileAccess BooleanDisallow local and piped files to access other local files.
StopSlowScript BooleanStop slow running javascript.
DebugJavascript BooleanForward javascript warnings and errors to the warning callback.
ErrorHandling StringHow should we handle obejcts that fail to load, must be either "ignore", "abort" or "skip"
Proxy StringString describing what proxy to use when loading the object.


ToPdfConfig_HeaderFooter - Header/Footer configs.
NameTypeDescription
FontSize LongThe font size in point to use for the header, e.g. "13"
FontName StringThe name of the font to use for the header. e.g. "times"
LeftText, CenterText, RightText String The text to print in the [LEFT|CENTER|RIGHT] part of the header, Following replaces occur in each of texts:

* [page] Replaced by the number of the pages currently being printed

* [frompage] Replaced by the number of the first page to be printed

* [topage] Replaced by the number of the last page to be printed

* [webpage] Replaced by the URL of the page being printed

* [section] Replaced by the name of the current section

* [subsection] Replaced by the name of the current subsection

* [date] Replaced by the current date in system local format

* [time] Replaced by the current time in system local format
Line BooleanWhether a line should be printed (under the header) or (above the footer).
Spacing DecimalThe amount in point of space to put between the header and the content, e.g. "1.8". Be aware that if this is too large the header will be printed outside the pdf document. This can be corrected with the margin.top setting.
HtmlUrl StringUrl for a HTML document to use for the header.
